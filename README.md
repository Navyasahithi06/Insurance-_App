# 💙 Health Insurance Cost Prediction

A Machine Learning web application that predicts an individual's estimated health insurance cost based on personal and lifestyle information. The application is built using Streamlit and uses a trained Random Forest Regressor model to estimate insurance charges.

---

## 📌 Project Overview

Health insurance premiums are influenced by several factors such as age, BMI, smoking status, number of children, gender, and residential region. This application allows users to enter these details and instantly predicts the estimated health insurance cost using a trained Machine Learning model.

---

## 🚀 Features

- Simple and user-friendly interface
- Real-time insurance cost prediction
- Machine Learning-based prediction
- Automatic feature encoding

## ▶️ How to Run the Project

### 1. Clone the Repository

```bash
git clone https://github.com/yourusername/Health-Insurance-Cost-Prediction.git
```

### 2. Open the Project Folder

```bash
cd Health-Insurance-Cost-Prediction
```

### 3. Install Required Libraries

```bash
pip install -r requirements.txt
```

### 4. Run the Streamlit Application

```bash
streamlit run app.py
```
